# Security Policy

## Supported Versions

Only the latest major version of Nifra is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.x    | :white_check_mark: |

## Reporting a Vulnerability

We take the security of Nifra seriously. If you find a security vulnerability, please do not report it in public issues. Instead, please report it privately.

To report a vulnerability, email us at **security@nifra.dev**.

Please include:
- A description of the vulnerability and its potential impact.
- Detailed steps to reproduce the issue (code samples, screenshots, or curl commands are highly appreciated).
- Any details of potential remediations or workarounds.

We will acknowledge receipt of your report within 48 hours and work with you to coordinate a disclosure timeline and fix.
